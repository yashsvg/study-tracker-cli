# 📌 Study Tracker CLI (C)

A CLI-based task manager built in C implementing CRUD operations with file persistence.

---

## 🚀 Features
- Add tasks  
- View tasks  
- Mark tasks as completed  
- Edit tasks  
- Delete tasks  
- Persistent storage using file handling  

---

## 🛠 Tech Stack
- C
- File Handling
- Command Line Interface (CLI)

---

## ▶️ How to Run

```bash
gcc main.c -o tracker
.\tracker.exe
```
