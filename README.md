\# 📌 Study Tracker CLI (C)



A simple command-line based study/task tracker built in C with file persistence.



\---



\## 🚀 Features

\- Add tasks

\- View tasks

\- Mark tasks as completed

\- Edit tasks

\- Delete tasks

\- Data stored in file (persistent)



\---



\## 🛠 Tech Stack

\- C

\- File Handling

\- CLI (Command Line Interface)



\---



\## ▶️ How to Run



```bash

gcc main.c -o tracker
